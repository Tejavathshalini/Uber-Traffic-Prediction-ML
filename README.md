# Uber Traffic Volume Prediction using Machine Learning

This project analyzes historical traffic data to predict hourly traffic volumes at different road junctions.

The project was completed as part of a Mentor-Led Internship with Uber and upGrad.

## Project Objectives

- Identify peak traffic hours
- Understand factors influencing congestion
- Develop predictive models for traffic forecasting

## Dataset Features

- Junction ID
- Traffic Volume
- Temperature
- Rain indicator
- Humidity
- Hour of day
- Day of week
- Month
- Event indicators

## Models Used

- Linear Regression
- Random Forest Regressor

## Evaluation Metrics

- MAE
- RMSE
- R² Score

## Key Insights

- Peak congestion occurs during morning and evening rush hours
- Junction J2 experiences highest traffic
- Weather and events influence congestion patterns
